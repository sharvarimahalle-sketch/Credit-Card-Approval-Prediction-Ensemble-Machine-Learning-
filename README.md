## Credit Card Approval Prediction

📌 Overview
Predicts credit card approval status using Ensemble Machine Learning to help financial institutions automate credit risk assessment.
Dataset: 438,000+ customer records
Best Model: Random Forest Classifier
Accuracy: 87.7% | F1-Score: 93.1%

🛠️ Tech Stack
Python | Pandas | Numpy | Scikit-learn | Matplotlib | RandomUnderSampler

📊 What I Did
1. Data Preprocessing
Cleaned 438K+ records, handled missing values and duplicates
Encoded categorical variables (Gender, Education, Property ownership)
Applied feature engineering to create predictive variables

2. Handled Class Imbalance
Used RandomUnderSampler to balance approval/rejection classes
Prevented model bias toward majority class

3. Built Ensemble Models
Random Forest ( 87.7% accuracy)

4. Model Evaluation
Achieved 87.7% accuracy and 93.1% F1-score
Analyzed feature importance to identify key approval drivers
Generated confusion matrices

Key Takeaways
✅ Applied ensemble learning for financial risk prediction
✅ Handled severe class imbalance in real world data
✅ Built production ready ML pipeline with 87.7% accuracy
✅ Generated actionable insights using feature importance analysis
